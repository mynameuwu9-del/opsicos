# APK Downloads Directory

This directory contains the compiled Android APK files for the Opsicos app.

## Current Version
- **Version**: 1.0.0
- **File**: opsicos-latest.apk
- **Status**: Pending build

## Build Instructions

To build the Android APK:

1. Install Android Studio and Android SDK
2. Navigate to the `android-app` directory
3. Run the build script:
   ```bash
   ./build-android.sh
   ```

The APK will be automatically placed in this directory.

## Distribution

The APK is distributed through:
- Web download page: `/download-app`
- Direct link: `/android-app/downloads/opsicos-latest.apk`

## Security

Each APK is signed and includes a SHA-256 hash for verification.
